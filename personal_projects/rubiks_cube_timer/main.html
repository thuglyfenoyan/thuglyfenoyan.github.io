Rubik's Cube Timer

One of my biggest hobbies in Grade 9 was speedcubing, which is the art of solving various Rubik's cube-like puzzles in the least amount of time possible (My personal best on a regular 3x3 Rubik's Cube is 13.98 seconds). I had built quite a collection of puzzles over the years.

One of my best solves recorded on camera.

One boring day, I felt the urge to use my Python programming knowledge to build a Rubik's Cube timer with various functionalities. Some of these included:

  • modes for 2x2, 3x3, 4x4, 5x5 and 6x6 puzzles
  • calculating the cubing averages
  • accessing history
  • deleting solves and updating history by doing so
  • displaying statistics for each puzzle mode

The source code for this project can be found at my GitHub.
